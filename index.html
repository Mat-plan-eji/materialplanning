🔥 Panduan Setup Firebase

1. Buka console.firebase.google.com → Add project → beri nama (contoh: material-planner) → lanjutkan.

2. Di sidebar kiri → Build → Firestore Database → Create database → pilih mode Production → pilih region terdekat (asia-southeast2 = Jakarta).

3. Di Firestore → tab Rules → ganti isinya dengan:
rules_version = '2';
service cloud.firestore {'{'}
  match /databases/{database}/documents {'{'}
    match /{document=**} {'{'}
      allow read, write: if true;
    {'}'}
  {'}'}
{'}'}
4. Kembali ke Project Overview → klik ikon </> Web → beri nama app → klik Register App.

5. Copy nilai apiKey, projectId, dan appId dari config yang muncul → paste ke Settings di atas.

6. Klik Simpan & Hubungkan → tunggu status jadi hijau ✓
Free tier Firebase: 1 GB storage, 50.000 reads/hari, 20.000 writes/hari — lebih dari cukup untuk tim kecil.
